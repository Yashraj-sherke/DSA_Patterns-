📚 STL Containers — When to Use What
📚 Stack (LIFO)
Last-In First-Out. Like a stack of plates — you can only add/remove from the top.
#include <stack> stack<int> s; s.push(10); // add to top s.top(); // peek top s.pop(); // remove top s.size(); // count s.empty(); // is empty?
🚶 Queue (FIFO)
First-In First-Out. Like a line at a counter — first person in line gets served first.
#include <queue> queue<int> q; q.push(10); // add to back q.front(); // peek front q.back(); // peek back q.pop(); // remove front q.size(); q.empty();
🔄 Deque (Double-Ended)
Insert/remove from both ends in O(1). Like a two-sided queue.
#include <deque> deque<int> dq; dq.push_front(1); dq.push_back(2); dq.pop_front(); dq.pop_back(); dq[i]; // random access O(1)
🎯 Set (Sorted Unique)
Stores unique elements in sorted order. Uses Red-Black Tree.
#include <set> set<int> s = {3,1,2}; // → {1,2,3} s.insert(4); // O(log n) s.erase(2); // O(log n) s.count(3); // 0 or 1 s.find(3); // iterator
⚡ Priority Queue (Max Heap)
Always gives the largest element first. Uses a binary heap.
#include <queue> priority_queue<int> pq; // max heap pq.push(3); pq.push(7); pq.push(1); pq.top(); // → 7 (largest) pq.pop(); // removes 7 // Min heap: priority_queue<int,vector<int>, greater<int>> minPQ;
