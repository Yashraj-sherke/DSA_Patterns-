📚 Hashing Concepts — Quick Reference
🔑 What is Hashing?
Hashing maps data (keys) to fixed-size values (hash codes) using a hash function. This enables O(1) average-time access.
Key → hash_function → index → bucket
🏗️ Internal Structure
unordered_map uses a hash table with buckets (chaining).
map uses a Red-Black Tree (sorted keys).
Bucket[0] → {"apple",120} Bucket[2] → {"banana",40} → {"cherry",80} ← collision! Bucket[3] → {"mango",90}
⚡ Hash Function (C++)
C++ uses std::hash<T> for built-in types. Custom types need a custom hash functor.
std::hash<string> hasher; size_t h = hasher("hello"); // index = h % bucket_count

📊 Load Factor & Rehashing
Load Factor = size / bucket_count.
Default max = 1.0. Exceeding it triggers rehashing (array doubles, all entries re-indexed).
mp.load_factor(); // current mp.max_load_factor(); // default 1.0 mp.bucket_count(); // num buckets mp.reserve(100); // pre-allocate
