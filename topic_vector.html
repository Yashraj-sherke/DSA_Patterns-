📚 Vector Internals — Quick Reference
📐 What is a Vector?
A dynamic array that can grow/shrink at runtime. Stores elements in contiguous memory for cache-friendly access.
#include <vector> vector<int> v = {1, 2, 3}; v.push_back(4); // grows automatically
🧮 Size vs Capacity
size() = number of elements stored.
capacity() = allocated memory slots.
When size exceeds capacity → reallocation (doubles capacity).
v.size(); // elements count v.capacity(); // allocated slots v.reserve(100); // pre-allocate v.shrink_to_fit(); // free excess

🏗️ Memory Layout
Vector stores a pointer to a heap-allocated array, plus size & capacity metadata.
Stack: [ptr | size=3 | cap=4] ↓ Heap: [10][20][30][ _ ] 0 1 2 (unused)
🎯 When to Use Vector
✅ Need dynamic size
✅ Frequent random access (v[i])
✅ Mostly add/remove at the end
❌ Frequent insert/delete in the middle → use list
❌ Need unique elements → use set
